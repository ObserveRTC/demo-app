ObserveRTC Demo Application
---

## Clone

```bash
git clone https://github.com/ObserveRTC/demo-app.git
```

## Install and Build

1. Install the server 

```bash
cd server && yarn
```

2. Install the webapp

```bash
cd webapp && yarn
```

## Run

1. Start server

```bash
cd server

node dist/main.js
```

2. Start the webapp

```bash
cd webapp

yarn start
```
